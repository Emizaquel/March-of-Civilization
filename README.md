This is the codebase for the March of Civilization available here: https://emizaquel.com/projects/MoC/

If you want to run any version before Pre-release 0.6 yourself locally, you will need to spin up a local server due to cross-site scripting issues. The local python server (eg: https://realpython.com/python-http-server/) is how I test changes prior to a release/pre-release

0.6 and onward, you can simply open the MoC.html file in the archive
